# 项目核心知识库

## 项目目标
**航海志 (Navigator)** - 浏览器插件，将网页HTML内容智能提取并一键转换为EPUB电子书格式。

## 核心功能
1. **F1: 智能内容提取** - 启发式算法（文本密度 + DOM 结构分析）提取网页正文，移除广告/导航等干扰
2. **F2: 图片本地化与优化** - 自动发现、下载、压缩页面图片并嵌入EPUB
3. **F3: EPUB 生成与下载** - 生成 EPUB 3.0（兼容2.0），含元数据、目录、样式
4. **F4: 用户交互流程** - Popup窗口（400×500px），预览→转换→下载

## 技术栈
- **浏览器环境**: Chrome/Edge/Firefox (Manifest V3)
- **前端框架**: 原生 JavaScript（轻量级）
- **EPUB 生成**: epub-gen-memory 或 jsepub
- **DOM 分析**: 自实现启发式算法（基于 Readability.js 思路）
- **图片处理**: Canvas API（压缩）+ 原生 fetch API
- **打包工具**: Vite

## 核心共识
- 所有数据处理本地完成，不发送到服务器
- 代码模块化：内容提取、图片处理、EPUB 生成分离
- UI 支持中英文
- 性能目标：内容提取 <2s，总流程 <45s
